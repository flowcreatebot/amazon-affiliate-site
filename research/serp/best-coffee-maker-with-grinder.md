## Repeatable structure patterns to copy

### 1) Title angle patterns winning the click
- **Year + tested volume**: “We Tested 20+…”, “Best … of 2026”
- **Outcome promise**: “These Were the 5 Best”, “Ultimate Fresh Brew Guide”
- **Use-case qualifier**: built-in grinder, single-serve, drip vs espresso, budget

**Template implication:** include year, test/selection proof signal, and use-case in title.

### 2) Section order pattern (most common)
1. Short intro + who this is for
2. **How we tested / methodology**
3. “Top picks” snapshot (table/cards)
4. Individual product blocks (Best Overall, Budget, etc.)
5. Buying guide (grinder type, cleaning, brew volume, noise)
6. FAQ

**Template implication:** lock this order so every money page satisfies both skim-readers and deep readers.

### 3) Comparison table style pattern
Most pages lead with a fast-scan table with:
- Product name
- “Best for” tag
- Grinder type (burr/blade)
- Cup capacity
- Notable setting count (grind + strength)
- Link/CTA column (check price)

**Template implication:** table must appear before first long review block.

### 4) Product block pattern
Top performers include:
- “What we like / what we don’t” or pros/cons
- Key specs (dimensions, settings, carafe type)
- A practical verdict (“who should buy it”)

**Template implication:** each product card needs quick pros/cons + key specs + buyer fit sentence.

### 5) FAQ / PAA pattern observed
PAA themes showing repeatedly:
- Are pricier coffee makers worth it?
- Burr vs blade: what matters?
- How to make coffee taste better at home?
- Best choice for black coffee / clean cup

**Template implication:** FAQ should target value tradeoff + grinder quality + brew quality outcomes.

## Intent gaps competitors leave open (quick opportunities)

1. **Maintenance friction is under-specified**
   - Competitors mention cleaning, but few score “maintenance burden” clearly.
   - Opportunity: add per-machine “cleanup effort” and “chute clog risk” rows.

2. **Single-serve + built-in grinder intent is underserved**
   - SERP related queries include strong demand for “single serve coffee maker with grinder built-in”.
   - Opportunity: dedicated page rather than burying one model inside a broad roundup.

3. **Burr vs blade education is scattered**
   - Many pages mention it but don’t quantify impact by brew method.
   - Opportunity: mini decision matrix (drip/espresso/single-serve + recommended grinder type).

4. **Price-band decision support is weak**
   - Users still need “under $150 / $300 / premium” clear picks.
   - Opportunity: price-tier callouts in top table and section anchors.

## Recommended on-page blueprint for this cluster

- H1: include keyword + year + “tested/reviewed” proof token
